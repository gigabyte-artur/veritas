# Veritas 📱✨

Мобильный тренажер для запоминания, разработанный на платформе **1С:Предприятие**.

## 📖 О проекте
`Veritas` — это приложение-тренажер, предназначенное для эффективного запоминания информации. Приложение использует проверенные методики интервального повторения и адаптивного обучения, позволяя быстро усваивать языки, профессиональные термины, формулы или любые другие данные прямо с мобильного устройства.

## 🚀 Особенности
- ✅ Адаптивный интерфейс под мобильные экраны
- 🧠 Гибкая настройка карточек и режимов повторения
- 📊 Встроенная аналитика прогресса и статистика запоминания
- ☁️ Возможность синхронизации баз данных (если реализовано)
- 🔒 Работа в оффлайн-режиме без подключения к интернету
- 🛠️ Полная интеграция с мобильной платформой 1С

## 🛠️ Установка и запуск
> Для работы требуется установленное мобильное приложение **1С:Предприятие**.
